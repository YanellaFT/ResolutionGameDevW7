# ResolutionGameDevW7
vanilla js game following the Week 7 tutorial for Resolution Game Dev but with own styling twist that took way to long...

use arrow or WASD keys to move player, now snowman, to catch the falling snowflakes. Dont let them fall past you because then you will lose a life!
(this sounded very appealing after one week of 90+ degree weather!)

AI usage: I used AI to help design the snowflakes bc could not figure out how to get the lines to look like a snowflake
